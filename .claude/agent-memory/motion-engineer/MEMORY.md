## Motion architecture (single sources) — prior build 2026-07-07 (deleted in reset; rebuild to this spec)
- GSAP registered ONLY in `src/lib/gsap.ts` (registers ScrollTrigger, defaults `ease: "power4.out", duration: 0.8`; exports `gsap`, `ScrollTrigger`). No other file imports `gsap` or `gsap/ScrollTrigger`.
- **Refresh-at-top contract (2026-07-19):** `lib/gsap.ts` sets `history.scrollRestoration = "manual"` **before** `registerPlugin(ScrollTrigger)`, and `main.tsx` calls `window.scrollTo(0, 0)` at module scope (router `scrollRestoration` option removed — it had no other job). Gotcha: ScrollTrigger snapshots `history.scrollRestoration` at init and re-writes the snapshot on EVERY `refresh()` (`_clearScrollMemory`), and import hoisting runs `lib/gsap` before `main.tsx`'s body — so setting "manual" anywhere after registration silently reverts to "auto" at the first refresh (verified live).
- Lenis instantiated ONLY in `src/providers/SmoothScrollProvider.tsx` (lerp 0.09, wheelMultiplier 1, smoothWheel, syncTouch false): `instance.on("scroll", () => ScrollTrigger.update())` (arrow wrapper — direct handoff fails strict TS), `gsap.ticker.add(t => lenis.raf(t*1000))`, `lagSmoothing(0)`; exports `LenisContext`; destroys + removes ticker on cleanup. Under reduced motion Lenis is NEVER created → `useLenis()` (in `src/hooks/useLenis.ts`) returns null.
- No manual resize-refresh listener — ScrollTrigger auto-refreshes on resize. Route-change refresh not wired (single-page site); add only if multi-page ships.
- Provider composition: `src/providers/AppProviders.tsx` = ThemeProvider(defaultTheme="dark") → TooltipProvider → SmoothScrollProvider. `main.tsx` = StrictMode → AppProviders → RouterProvider.
- Every animation in `useGSAP(() => {…}, { scope: ref, dependencies: [...] })`. Returning a function from the useGSAP callback registers a context cleanup (used for `split.revert()`). `invalidateOnRefresh: true` on scrubs.

## Primitive APIs (src/components/common/, all exported from barrel index.tsx)
- `RevealText` — `{ children, mode? = "lines", as? = "div", className?, delay?, stagger?, trigger?: RefObject|string, start? = "top 80%", end?, scrub?: boolean|number, blurFrom? = 0, veiledOpacity? = 0 }` (extended 2026-07-18). `scrub` undefined → shipped once-reveal (yPercent clip rise, `once:true`). `scrub` set → scrubbed REVERSIBLE veiled blur→clear variant, re-split + rebuilt on refresh (mechanics: [about-refine.md](about-refine.md)). **GOTCHA (fixed 2026-07-19): after an onRefresh rebuild the scrub tl MUST `tl.progress(self.progress)`** — a refresh past the trigger end leaves forward progress clamped at 1, the playhead never moves, and the fresh splits stay veiled forever (symptom: resize at page bottom re-blurred the About bio permanently). **GOTCHA: a parent's RefObject is still `null` in this child's layout effect** (React attaches refs walking up) — ancestors pass a SELECTOR (`trigger="#about"`). **split-type caveat: nested spans line-group as ONE atomic unit** — plain text only; mixed-face statements use React-owned word spans (StatementWords pattern).
- `ParallaxImage` — `{ src, alt, width?, height?, aspect? (className), parallax?: ParallaxConfig, withScrim?, className? }`. Figure clip `inset(100% 0 0 0)→inset(0)` (1.2s, once) + img `yPercent from→to` scrub. Raw `<img>` with `scale-[1.2]` headroom (common/ exempt; existing `Image` skeleton wrapper fights the clip). Scrim = `bg-linear-to-b from-transparent from-40% to-ink/65`.
- `Marquee` — `{ children, speed? = 30 (s/loop), reverse?, pauseOnHover? = true, className? }`. Two `.marquee-track` flex tracks (2nd `aria-hidden`), `xPercent 0→-100` repeat -1; tween in a ref, paused/played by React pointer handlers.
- `MagneticButton` — `{ children, className?, strength? = 12 }`. `gsap.quickTo` x/y on wrapper; `.magnetic-label` span counter-moves ×-0.35; spring back `elastic.out(1, 0.4)`.
- `PathDraw` — `{ d, trigger?: RefObject<HTMLElement|null> (defaults own wrapper), strokeWidth? = 3.5 (number → SVG attr; STRING → inline CSS, enables "clamp(1.25rem,5vw,6.5rem)" viewport-tracking widths), className?, viewBox? = "0 0 100 100", preserveAspectRatio? = "none", scrub? = true, start? = "top 80%", end? = "bottom 60%" }`. Bold path draw (§7.2): dash math is `pathLength={1000}` normalized (set dasharray=dashoffset=1000 → tween dashoffset→0, ease none, scrubbed; has `revertOnUpdate: true`). **Two hard-won gotchas (2026-07-20 Craft build): (1) non-scaling-stroke renders dashes in SCREEN space — an anisotropically stretched viewBox tiles the dash into disconnected capsule segments, so scrub callers must pass a PIXEL-SPACE `d` + matching 1:1 viewBox (ProjectsSection measures its layer via ResizeObserver and scales normalized coords with `scalePathD`); (2) pathLength must be 1000, not 1 — GSAP's CSS layer rounds stroke-dashoffset to whole px, quantizing a 0–1 range into a binary step at 50% scrub.** `stroke="currentColor"` + wrapper `cn("text-accent", className)`; `strokeLinecap="round"`; svg `aria-hidden focusable="false"`. Reduced motion: early return → renders fully drawn. Wired into 04 Projects (thread + finale); Journey rail next.
- `ChapterEyebrow` — `{ index, label, className? }`, NOT animated. `font-mono text-eyebrow`, `text-accent` index, muted em-dash + uppercase label.
- `Cursor` — no props. 8px dot (`h-2 w-2 bg-paper`) + 40px ring; centered via `gsap.set({ xPercent:-50, yPercent:-50 })` (don't use Tailwind translate — gsap x/y would stack oddly). quickTo follow (dot 0.12s, ring 0.45s); `pointerover` → `closest("a, button, [data-cursor]")` → ring scale 1.6 for unlabeled targets. **Pill mode (2026-07-20)**: a `data-cursor` label morphs the ring via conditional `cn()` classes into an auto-width paper pill (`h-10 w-auto bg-paper px-5 whitespace-nowrap`, label `text-ink font-bold uppercase`, NO `mix-blend-difference` — pill must stay white on any bg) at scale 1 (scaling a sized pill blurs text; `w-auto` sizes it, GSAP never tweens width). Unlabeled circle: `h-10 w-10 border-paper/50 mix-blend-difference`, unchanged. Toggles `cursor-none` on `<html>` while mounted+enabled. Returns null on coarse pointer / reduced motion. Mounted in `__root.tsx`. Labels in use: rows `View Project`, preview panel `See Project` (04).
