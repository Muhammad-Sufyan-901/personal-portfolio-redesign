## Notable decisions

- **Award node dots use `bg-accent`**, work/education `bg-faint` — three 10px dots across the page stay within "accent as scalpel" and give awards quiet distinction without color-coding text.
- **Entry title uses `Box as="h3"` + token classes**, not `Heading level={3}`: the Heading default variant's `md:text-3xl` survives the cn merge over `text-item` (responsive prefix is a different twMerge group), which would override the fluid token at md+. Same pattern as PillarBlock's title.
- Section owns all motion via hook classes (`.journey-rail`, `.journey-entry`); `JourneyEntry` stays purely presentational. Skipped a separate dot `scale 0→1` tween — the entry-level `autoAlpha` already covers the dot (coordinator marked it optional).
- Rail scrub: `trigger: ".journey-timeline", start "top 80%", end "bottom 60%", scrub: true, ease: "none"`; entries: `autoAlpha 0→1, y 24→0, start "top 85%", once: true` via `forEach` (no `ScrollTrigger.batch` needed for 9 static items).
