# Remove boilerplate auth/dashboard/register

- **Date:** 2026-07-06
- **Author:** main
- **Type:** refactor
- **Chapter/Area:** repo cleanup (pre-redesign)

## Summary
Removed the enterprise-boilerplate leftovers the portfolio doesn't need: the auth feature (login/register/logout), the dashboard feature, the `_auth`/`_protected` route trees, and their plumbing (auth store, route guards, axios layer, API envelope types). Routing is now just `/` → HomePage. This leaves a clean base for the motion-first redesign.

## Files touched
- `src/features/auth/**`, `src/features/dashboard/**` — deleted
- `src/routes/_auth/**`, `src/routes/_protected.tsx`, `src/routes/_protected/**` — deleted; `routeTree.gen.ts` regenerated by the router Vite plugin
- `src/middlewares/authMiddleware.ts`, `src/store/useAuthStore.ts`, `src/lib/axios.ts`, `src/config/env.ts`, `src/models/user.model.ts`, `src/types/api.type.ts` — deleted (auth-only plumbing)
- `src/features/home/` — demo content (HeroSection, FeatureCards, data, types) deleted; `pages/HomePage.tsx` rewritten as a minimal shell; `components/`, `data/`, `types/` kept empty via `.gitkeep`
- `src/components/ui/*` — pruned to `button.tsx` + `tooltip.tsx` (58 unused shadcn files removed; re-addable via `npx shadcn add`)
- `src/components/common/index.ts` — new barrel so features import from `@/components/common`
- `src/hooks/useMobile.ts` — deleted (only consumer was `ui/sidebar.tsx`)
- `src/main.tsx` — removed `QueryClient`/`QueryClientProvider`
- `index.html` — dropped boilerplate SEO meta; title → "Muhammad Sufyan"
- `eslint.config.js` — scoped `react-refresh/only-export-components: off` to `src/components/ui/**` (shadcn exports cva variants alongside components)
- `package.json` — uninstalled: axios, js-cookie (+types), zod, react-hook-form, @hookform/resolvers, zustand, @tanstack/react-query (+devtools), recharts, @tanstack/react-table, date-fns, embla-carousel-react, input-otp, cmdk, vaul, react-day-picker, react-resizable-panels, sonner, next-themes

## Notable decisions
- Pruned unused shadcn ui files + their orphaned deps (user-approved) instead of keeping all of `ui/` — any component is one `npx shadcn add` away.
- Kept `button`/`tooltip` as the baseline interactive controls per house rules; `TooltipProvider` stays in `main.tsx`.
- HomePage placeholder shows only the name + a neutral line — no invented content (PRD transcription happens in the redesign).
