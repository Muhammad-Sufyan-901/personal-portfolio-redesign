## Notable decisions
- Initial-hide is done with `gsap.set` inside `useGSAP` (never CSS) — JS-dead or reduced-motion paths always show static content. Pre-`preloaderDone` the hero stays set-hidden so the curtain wipe reveals a blank stage, not a flash.
- `revertOnUpdate: true` on the hero hook: the `preloaderDone` flip reverts the first context (including `split.revert()`) and re-splits clean markup.
- Name split uses `types: "chars"` only — the block word spans already isolate lines and carry `overflow-hidden`, so no line/word wrapping pass is needed (split-type ≥0.3 handles nested spans).
- Header under reduced motion is solid always (no scroll-linked state). MobileMenu close is instant (unmount) — exit tween deferred to QA taste.
- Scroll-spy active-nav-dot NOT built — needs the other sections' anchors to exist; noted in agent memory.
